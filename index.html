Tags aprendidas no curso

Índices

  1. Cabeçalhos: <h1> até <h6>
  2. Paragrafo: <p>
  3. Listas ordenadas: <ol> e <li>
  4. Listas não ordenadas: <ul> e <li>
  5. Small: <small>
  6. Sublinhado e itálico: <u> e <i>
  7. Strong: <strong>
  8. Linha horizontal: <hr>
  9. Subscrito e sobrescrito: <sub> e <sup>
  10. Citaçao em bloco: <blockquote>

Cabeçalhos

(Voltar)

São usadas para criar títulos e subtítulos com diferentes níveis de importância.

Exemplo de código:
<h1>Título principal</h1>
<h2>Subtítulo nível 2</h2>
<h3>Subtítulo nível 3</h3>

Exemplo visual:

<h1> É o título mais importante, geralmente o título principal da página.

<h6> É o menos importante, para subtítulos menores. Os navegadores exibem esses títulos com tamanhos diferentes e eles ajudam na organização e SEO.

Paragrafo

(Voltar)

O Paragrafo é usado para agrupar texto em blocos. É uma das tags mais comuns em HTML.

Exemplo de código

<p>Este é um parágrafo.</p>

Exemplo visual

"HTML é a linguagem base da web, usada para estruturar conteúdos como textos, imagens, links, listas, títulos e elementos essenciais de uma página."

Listas ordenadas

(Voltar)

<ol>: Lista numerada, usada quando a ordem importa (ex: passos).
<li>: Define cada item da lista.

Exemplo de código

<ol>
<li>Primeiro passo</li>
<li>Segundo passo</li>
</ol>

Exemplo visual

  1. Primeiro passo
  2. Segundo passo


Lista não ordenada

(Voltar)

<ul>: Lista com marcadores, para itens sem ordem específica.
<li>: Define cada item da lista.

Exemplo de código

<ul>
<li>Item um</li>
<li>Item dois</li>
</ul>

Exemplo visual

  • Item um
  • Item dois


Small

(Voltar)

A tag <small> é usada para exibir texto em tamanho menor, é usada para diminuir visualmente o tamanho da fonte de um texto. Ela é útil para mostrar notas de rodapé, observações, direitos autorais ou comentários secundários no conteúdo.

Exemplo de código

<small>Este é um texto pequeno.</small>

Exemplo visual

Este é um texto pequeno.


Sublinhado e itálico

(Voltar)

<u>: Sublinha o texto, usado para destacar, é apenas visual, ou seja, não indica nenhum significado especial..
<i>: Coloca o texto em itálico, usado para ênfase ou citações.

Exemplo de código

<u>Texto sublinhado</u>
<i>Texto em itálico</i>

Exemplo visual

Texto sublinhado
Texto em itálico


Strong

(Voltar)

A tag <strong> é usada para destacar texto com ênfase forte, geralmente exibido em negrito.
É usada para indicar que o texto é importante ou deve ser lido com atenção.

Exemplo de código

<strong>Texto em negrito</strong>

Exemplo visual

Texto em negrito


Linha horizontal

(Voltar)

A tag <hr> é usada para criar uma linha horizontal, que pode ser usada para separar seções de conteúdo.
É um elemento vazio, ou seja, não tem conteúdo entre tags de abertura e fechamento.

Exemplo de código

<hr>

Exemplo visual


Subscrito e sobrescrito

(Voltar)

<sub>: Coloca o texto abaixo da linha de base, usado para subscritos (ex: fórmulas químicas).
<sup>: Coloca o texto acima da linha de base, usado para sobrescritos (ex: potências).

Exemplo de código

<sub>Texto subscrito</sub>
<sup>Texto sobrescrito</sup>

Exemplo visual

Fórmula da água: H2O

Área do quadrado: lado2


Citação em bloco

(Voltar)

A tag <blockquote> é usada para exibir citações longas ou blocos de texto citados de outras fontes.
Ela geralmente é usada para destacar uma citação ou um trecho de texto que não é do autor da página.

Exemplo de código

<blockquote>Esta é uma citação em bloco.</blockquote>

Exemplo visual

Esta é uma citação em bloco.
